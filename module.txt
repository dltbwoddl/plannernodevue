#데이터 가져오는 함수
home에서 단기목표별로 그날에 맞는 todolist가져오는 함수
home에서 장기목표를 가져오는 함수
중기목표 페이지에서 장기목표이름을 인자로 받아 해당 중기목표 가져오는 함수
단기목표 페이지에서 중기목표이름을 인자로 받아 해당 단기목표 가죠오는 함수
todolist 페이지에서 단기목표이름을 인자로 받아 해당 todolist 가져오는 함수

#데이터 수정하는 함수
/home/modify에서 변경사항 저장을 눌렀을 때 리스트에 나온 데이터 대로 장기목표를 수정하는 함수.
/:장기목표이름/modify에서 변경사항 저장을 눌렀을 때 리스트에 나온 데이터 대로 중기목표를 수정하는 함수.
/:장기목표이름/:중기목표이름/modify 위와 같다.
/:장기목표이름/:중기목표이름/:단기목표이름/modify 위와 같다.(단기목표이름을 가진 테이블을 수정하면 된다.)
(변경된 부분만 수정하는 함수 만들기.)
